ДЗ-1 
Linux terminal (GitBash) commands

1) Посмотреть где я.  == pwd
2) Создать папк. == mkdir group_27
3) Зайти в папку. == cd group_27
4) Создать 3 папки. ==  mkdir folder_1 folder_2 folder_3
5) Зайти в любоую папку. == cd folder_1
6) Создать 5 файлов (3 txt, 2 json). == touch file_1.txt file_2.txt file_3.txt file_4.json file_5.json
7) Создать 3 папки. == mkdir folder_4 folder_5 folder_6
8. Вывести список содержимого папки. ==  ls -la
9) + Открыть любой txt файл. == vim file_1.txt
10) + написать туда что-нибудь, любой текст. == нажать клавишу 'i', затем ввести текст
11) + сохранить и выйти. == нажать клавишу 'Esc', затем ':' и ‘wq’
12) Выйти из папки на уровень выше. == cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. == 
 mv {file_1.txt,file_2.txt} folder_4 - если я нохусь в текущей папке;
 mv folder_1/{file_1.txt.file_2.txt} folder_1/folder_4/ - если я нахожусь в любой другой папке, например, в group_27
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. == 
 cp {file_3.txt,file_4.json} folder_5 - если я нохусь в текущей папке;
 cp ../{file_3.txt,file_4.json} ../folder_5/ - если я нахожусь в любой другой папке, например, в folder_4
15) Найти файл по имени. == find . –name file_1.txt
16) Просмотреть содержимое в реальном времени. == tail –f /file_1.txt - отображает все новые строки в файле;
tail -f file_1.txt | grep vika - отображает новые строки в файле, содержащие конкретное значение, например, "vika"

17) вывести несколько первых строк из текстового файла. == head -3 file_1.txt
18) вывести несколько последних строк из текстового файла. == tail -3 file_1.txt
19) просмотреть содержимое длинного файла. == less -3 file_1.txt
20) вывести дату и время. == date
